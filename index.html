Dev#Bilac

SEALP - Software de Ensino-Aprendizagem de Lógica de Programação

O objetivo do projeto é construir um software em três ambientes (desktop, web e mobile), utilizando a plataforma JAVA como plano de fundo, que irá apresentar um jogo virtual no qual os usuários podem aprender e/ou aprimorar seus conhecimentos sobre lógica de programação.

Para mais informações sobre o projeto entre em contato com os professores:
Prof. Esp. André Olímpio - andre.olimpio@bilac.com.br

MIT APP Inventor

Workshop de como fazer um aplicativo usando a lógica!

No Dia da Engenharia, nós alunos do Bilac junto com a professora Angela, fomos convidados a dar workshop de programação no ITA para alunos do Mariotto e do Alpha Lumen. Usamos a plataforma do APP Iventor, que é um site com ferramentas que possibilita fazer aplicativos móveis usando somente a lógica de programação.

Você pode saber mais sobre o MIT APP Inventor aqui.

Semana Acadêmica

Semana Acadêmica do Bilac.
